Call flush for endl if E\-N\-D\-L\-\_\-\-C\-A\-L\-L\-S\-\_\-\-F\-L\-U\-S\-H is nonzero

The standard for iostreams is to call flush. This is very costly for \hyperlink{class_sd_fat}{Sd\-Fat}. Each call to flush causes 2048 bytes of I/\-O to the S\-D.

\hyperlink{class_sd_fat}{Sd\-Fat} has a single 512 byte buffer for S\-D I/\-O so it must write the current data block to the S\-D, read the directory block from the S\-D, update the directory entry, write the directory block to the S\-D and read the data block back into the buffer.

The S\-D flash memory controller is not designed for this many rewrites so performance may be reduced by more than a factor of 100.

Set nonzero to use Serial (the \hyperlink{class_hardware_serial}{Hardware\-Serial} class) for error messages and output from print functions like ls().

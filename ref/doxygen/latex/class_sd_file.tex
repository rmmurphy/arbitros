Write data to an open file.

\begin{DoxyNote}{Note}
Data is moved to the cache but may not be written to the storage device until \hyperlink{class_sd_base_file_a3eb7f41182c04388c951db242c13f845}{sync()} is called.
\end{DoxyNote}

\begin{DoxyParams}[1]{Parameters}

\hline
\mbox{\tt in}  & {\em nbyte} & Number of bytes to write.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
For success \hyperlink{class_sd_file_ac82b138293686567ab5dfbe3ba1fa1ee}{write()} returns the number of bytes written, always {\itshape nbyte}. If an error occurs, \hyperlink{class_sd_file_ac82b138293686567ab5dfbe3ba1fa1ee}{write()} returns -\/1. Possible errors include \hyperlink{class_sd_file_ac82b138293686567ab5dfbe3ba1fa1ee}{write()} is called before a file has been opened, write is called for a read-\/only file, device is full, a corrupt file system or an I/\-O error. 
\end{DoxyReturn}

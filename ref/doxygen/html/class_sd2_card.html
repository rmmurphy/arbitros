bool setSckRate ( uint8_t  sckRateID)

Set the SPI clock rate.

Parameters
[in]sckRateIDA value in the range [0, 14].

The SPI clock divisor will be set to approximately

(2 + (sckRateID & 1)) << ( sckRateID/2)

The maximum SPI rate is F_CPU/2 for sckRateID = 0 and the rate is F_CPU/128 for scsRateID = 12.

Returns
The value one, true, is returned for success and the value zero, false, is returned for an invalid value of sckRateID.

bool(* pf_funPtr)(t_DEVHANDLE t_consoleHndl, int8_t *pc_buff, t_consoleTokHndl *pt_tokHndl)

Pointer to the user-space console handler configured during module initialization (#arb_consoleInit). The console thread (arb_console) passes control to this function after determining that a command (entered over the terminal) is not part of the Arbitros basic kernel command set.

t_DEVHANDLE t_consoleHndl

Handle to the console thread created during module initialization (#arb_consoleInit). Handles are the primary mechanism for linking a thread (i.e. arb_console)–through the kernel–to the particular device driver its trying to access.

t_THRDHANDLE t_consoleThread

Handle to the console thread created during module initialization (#arb_consoleInit).

t_DEVHANDLE t_sdHndl

Handle to the fat32 SD card driver created during module initialization (#arb_consoleInit). This handle provides a way of exposing file system functionality–such as directory searching, file deletion, and reading the contents of a file–to an external user via the terminal command window by way of the arb_console thread.
